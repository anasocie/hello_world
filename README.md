# 🌍 Polyglot Hello World

> *"A journey of a thousand lines begins with a single 'Hello, World!'"*

A collection of "Hello, World!" implementations in multiple programming languages. Built to explore syntax, practice coding, and — of course — keep that GitHub contribution graph lush and green 🌱

---

## 🚀 Purpose

- Learn and revisit the syntax of different programming languages
- Have a quick reference when starting new projects
- Maintain consistent daily study habits through small contributions
- **Bonus:** Keep the GitHub streak alive 🔥
